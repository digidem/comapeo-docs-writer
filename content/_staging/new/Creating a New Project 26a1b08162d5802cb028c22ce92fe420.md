## Starting a New Project

In the case where you want to share any observations from My Solo Project, but would rather collaborate with another device on a separate project, then Start a New Project.

<aside>
ðŸ‘£

### Step by Step

***Step 1:*** Go to the :three-line-menu-black:**Menu**.

***Step 2:*** Tap on :app-icon-participant: **Collaborate**.

***Step 3:*** Select :app-icon-coordinator: **Start New Project.**

***Step 4:*** Add a **project name** and continue

***Step 5:*** Select a **Project Statistics** option. This can be changed in Coordinator Tools

***Step 5:*** Your new project is ready! 

</aside>
