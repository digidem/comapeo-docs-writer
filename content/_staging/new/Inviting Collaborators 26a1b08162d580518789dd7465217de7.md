# Inviting Collaborators

![](./images/extracted_0406c3b710f4.png)

## Why invite collaborators to your projects?

When mapping on your own, there are three ways to start collaborating with others using CoMapeo.

- Inviting a Collaborator to My Solo Project
- Starting a New Project
- Joining an Existing Project

---

## Inviting a Collaborator when there is Only One Project

When mapping on your own you can gather a limitless amount of observations. To invite a teammate to join the work that  you already started on, the project you are using needs to be converted into a collaboration

### Step by Step

***Step 1:*** Go to the :three-line-menu-black:**Menu**.

***Step 2:*** Tap on :app-icon-participant: **Collaborate**.

***Step 3:*** Select **Invite Devices**

![photo_2025-11-26_20-23-47.jpg](Inviting%20Collaborators/photo_2025-11-26_20-23-47.jpg)

***Step 4:*** Change the **project name** and continue

***Step 5:*** Your project is now ready to invite collaborators
